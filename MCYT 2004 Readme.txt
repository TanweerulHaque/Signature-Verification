Off-Line Signature:

	Data: 75 users x ( 15 genuine_signatures/user + 15 forgeries/user ) = 2250 signature images

Naming convention:

  * GENUINE SIGNATURES:

	- MCYT user number (4 digits) + "v" (stands for "verdadera" == "genuine")
	- MCYT signature number (2 digits).

		e.g.: "0002v00"


  * SKILLED FORGERIES:

	- MCYT user number of the forger (4 digits) + "_"
	- MCYT user number being forged (4 digits) + "f" (stands for "forgery")
	- MCYT signature number (2 digits).

		e.g.: "0003_0002f00"
